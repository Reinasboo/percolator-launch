## Summary

Removes the dead `ORACLE_BRIDGE_URL` import from `app/app/api/airdrop/route.ts`.

## Problem

The constant:
```ts
const ORACLE_BRIDGE_URL = process.env.ORACLE_BRIDGE_URL ?? "http://127.0.0.1:18802";
```
was declared but never used. A code comment indicated it was removed in favor of the `markets_with_stats` Supabase view. The dead constant misleads code readers about the price data source and should be removed.

## Fix

Removed the unused constant.

## Audit Reference

Fixes audit finding **N6 (Informational)** — dead `ORACLE_BRIDGE_URL` import in `airdrop/route.ts`.
